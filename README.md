# banchan

Collection of small Python utilities

## Examples

- Celery's `signal` utility is cloned from Django's `dispatch` module
- Decorators such as `cached_property`, `memoized` are everywhere

## Dependencies

- Pillow (for `image`)
- PyCrypto (for `crypt`)
- simplejson (for `json`)
- datrie (for `emit`)
- rtree (for `geo`)
- dateutil (for `date`)
